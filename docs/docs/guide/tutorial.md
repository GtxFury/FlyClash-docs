# 使用教程

完成 FlyClash Android 的下载与安装后，只需几个简单步骤就能快速接入机场服务。本教程会带你完成订阅导入、开启系统代理以及选择节点的流程，并配合示意图展示关键操作位置。

## 导入订阅链接

### 一键导入

大多数机场面板都提供一键导入 Clash 的入口。以 v2board 面板为例，只需点击首页的 `一键订阅导入 Clash` 按钮，FlyClash 会自动识别并写入你的订阅配置。

![v2board 面板的一键导入按钮展示](/tutorial/quick-import.png)

### 手动复制订阅导入

如果面板没有一键按钮，也可以在 FlyClash 内手动添加：

1. 进入 `配置` 页面并点击右上角的 `+` 图标。
2. 在弹出的菜单中选择第二项 `URL`。
3. 将机场面板提供的订阅链接粘贴到`订阅URL`输入框中，然后点击 `提交`。

![配置页手动导入订阅的三个步骤](/tutorial/manual-import.png)

## 打开开关，开始使用

回到首页仪表盘，点击右上角的启动。首次启动时，安卓系统会提示需要授权vpn权限，这一步只需要打开一次即可生效。

![首页仪表盘点击启动，开启服务](/tutorial/toggle-proxy.png)

在仪表盘点击启动，开启服务

![点击同意授权](/tutorial/allow.png)

在弹出的请求弹框中点击允许

## 选择节点

在 `代理` 页面，可以手动切换你想使用的节点。默认情况下会自动为你挑选最佳节点，也可以点击列表改成你需要的代理，实现自定义分流。

![代理页节点列表示意，支持自动或手动选择节点](/tutorial/proxy.png)

按照以上步骤，FlyClash 就已经完成基础配置，可以稳定使用了。接下来可以继续阅读进阶章节，了解策略分组、规则管理等高级功能。
