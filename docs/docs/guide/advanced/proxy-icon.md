# 代理组图标
FlyClash的代理页面支持代理组图标功能，支持以下两种格式，您可以通过下面的方法自定义代理组图标，丰富视觉体验。

### Clash Meta官方格式

FlyClash的代理页面支持Clash Meta icon字段，当配置文件中含有相关字段时，应用会自动下载并显示图标

`icon:"https://xxxx.png"`

### 应用内格式

您还可以通过FlyClash 代理页面的代理组图标功能手动添加图标，当图标规则匹配中时，所有代理组都将显示您设置的图标。

![](/tutorial/proxy-icon.png){style="max-width:50%;"}

进入代理页面，点击右上角进入代理组图标功能界面

![](/tutorial/proxy-icon2.png){style="max-width:50%;"}

点击右上角的添加按钮添加代理组图标

![](/tutorial/proxy-icon2.png){style="max-width:50%;"}

按照提示输入图标规则等信息

规则名称：用于区分规则的名字

正则表达式，用于匹配代理组名字的规则

例如：节点选择|Proxy (意思为如果配置中的代理组有节点选择或Proxy的代理组，将会匹配此图标)

图标类型：支持URL或Base64格式

图标数据：将图标直链或Base64格式的数据粘贴其中

优先级：数字越大优先级越高

点击保存即可完成添加规则

::: tip
两者优先级

当配置同时有icon字段，并开启了代理组图标，
规则并命中后将优先显示配置中的icon字段图标。
:::
