# 简介

FlyClash Android 是一款基于 Clash Meta 内核打造的安卓代理软件，兼顾简洁优雅的界面与强大的网络能力。

## 核心特性

- **广泛的协议支持**：支持先进规则体系与协议支持，快速适配常见订阅与策略分流场景。
- **简洁优雅的界面**：采用明快的视觉风格与扁平化布局，让常用功能触手可及，减少学习成本。
- **强大的管理能力**：支持多配置切换、节点分组、满足不同网络环境下的灵活需求。
- **贴心的交互细节**：内置状态提醒、错误指引与常用操作快捷入口，帮助你在移动端也能高效掌控代理服务。

## 我们的愿景

FlyClash 专注于为安卓用户提供可靠、易用的代理体验：

- 降低入门门槛，让初次接触代理软件的用户也能快速上手。
- 保持与 Clash Meta 的同步迭代，持续引入前沿能力。
- 通过社区反馈与开源生态不断完善功能，打造长期可用的移动端解决方案。
