# 脚本覆写
FlyClash Android 支持脚本覆写功能，您可以通过js脚本或者yaml轻松修改自定义您的配置文件

### 如何使用

进入到配置页面，点击右上角第一个的代码图标进入到脚本覆写页面。

![](/tutorial/profile.png){style="max-width:50%;"}

### 添加脚本
点击添加按钮进入脚本编辑界面

![](/tutorial/rewrite.png){style="max-width:50%;"}

将js脚本粘贴进框内，也可以点击右上角的链接图标在线下载。

![](/tutorial/rewrite2.png){style="max-width:50%;"}

您可以点击覆写范围卡片，选择要针对哪一个配置文件生效。

![](/tutorial/rewrite3.png){style="max-width:50%;"}

完成后点击保存，点击圆圈即可激活该覆写配置，取消同理。

![](/tutorial/rewrite4.png){style="max-width:50%;"}
