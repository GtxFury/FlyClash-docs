# 使用教程

完成 FlyClash Android 的下载与安装后，只需几个简单步骤就能快速导入配置。本教程会带你完成配置导入、开启代理以及选择节点的流程，并配合示意图展示关键操作位置。

## 导入订阅链接

### 一键导入

大多数代理提供商面板都提供一键导入 Clash 的入口。例如只需点击首页的 `一键订阅导入 Clash` 按钮，FlyClash 会自动识别并写入你的订阅配置。

![](/tutorial/quick-import.png)

### 手动复制订阅导入

如果面板没有一键按钮，也可以在 FlyClash 内手动添加：

1. 进入 `配置` 页面并点击右上角的 `+` 图标。
2. 在弹出的菜单中选择第二项 `URL`。
3. 将机场面板提供的订阅链接粘贴到`订阅URL`输入框中，然后点击 `提交`。

![配置页手动导入订阅的三个步骤](/tutorial/manual-import.png){style="max-width:50%;"}

## 打开开关，开始使用

回到首页仪表盘，点击右上角的启动。首次启动时，安卓系统会提示需要授权vpn权限，这一步只需要打开一次即可生效。

![首页仪表盘点击启动，开启服务](/tutorial/toggle-proxy.png){style="max-width:50%;"}

在仪表盘点击启动，开启服务

![点击同意授权](/tutorial/allow.png)

在弹出的请求弹框中点击允许

## 选择节点

在 `代理` 页面，可以手动切换你想使用的节点。您可以点击列表改成你需要的代理，实现自定义分流。

![代理页节点列表示意，支持自动或手动选择节点](/tutorial/proxy.png){style="max-width:50%;"}

按照以上步骤，FlyClash 就已经完成基础配置，可以正常使用了。接下来可以继续进阶教程，了解更多高级功能。
