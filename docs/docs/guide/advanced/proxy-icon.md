# 自定义仪表盘

FlyClash的仪表盘采用模块化设计，您可以根据需要自定义仪表盘页面，添加或者移出自己需要的小组件，自定义排序。

### 进入编辑模式

点击右上角的`铅笔图标` 进入编辑模式。 

![](/tutorial/dashboard-edit.png)

### 添加添加小组件

点击右上角的`添加` 添加小组件卡片。 

![](/tutorial/add-card.png)

## 排序卡片

长按住卡片的标题区域，卡片微微弹起，拖拽卡片即可调整卡片顺序

![](/tutorial/switch-card.png)

## 删除卡片

点击白色的小垃圾筒图标，即可删除卡片
